# Personal Website as Productized Application

My goal in building this website is to treat it as a real application, which requires modern DevOps practices. To that end, the following are already handled:

## Hosting
The website is currently hosted on GitHub Pages in a containerized environment.

## Live Demos
Projects displayed in the website are hosted elsewhere, and accessed via HTTP requests.

## CI
GitHub Actions allows testing the website on every push to a branch.
